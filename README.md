# KeebView
COSC 175 Final Project. Gets posts from r/customkeyboards

## Requirements
| Dependency | Version |
| ---------- | ------- |
| Python     | 3+      |
| Flask      | 2.0.2   |
| praw       | 7.4.0   |

## Installation
`$ pip install -r requirements`

## Environment variables
`FLASK_APP=keebView`  
`FLASK_RUN_HOST=0.0.0.0`   
`FLASK_RUN_PORT=3000`   

## Running locally
`$ flask run`
